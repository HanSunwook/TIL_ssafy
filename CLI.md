# CLI

그래픽을 통해 사용자와 컴퓨터가 상호작용하는 방식

효율성,정밀제어,자동화,표준환경

> (.): 현재 디렉토리\
(..): 현대의 상위 디렉토리

(touch): 파일생성\
(mkdir): 폴더생성\
(mv): (mv 파일 폴더) 폴더로 파일 이동\
(cd): 폴더로 이동\
(ls): 현재 파일\
(code 폴더/파일): 폴더의 파일 열기\
